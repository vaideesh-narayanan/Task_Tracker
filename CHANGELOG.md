# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-11
### Added
- Initial Release.
- Offline-first Task tracking functionality with Room local database.
- Settings page with theming (Light/Dark/Accent color).
- Reminder mechanism using local notification system.
- Export and Import task data cleanly.
- Categories and sorting system.
- Backup and restore feature via JSON structure.
- Recycle Bin functionality for accidentally deleted elements.
